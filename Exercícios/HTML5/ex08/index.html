Principais formatações

Negrito

Existem duas formas de usar o negrito, uma não-semantica e outra semantica, a forma não-semântica é com a tag <b>,
a forma semântica se escreve com a tag <strong>

Exemplo: este pedaço está com negrito na sua forma semâtica

Itálico

Já em itálico temos a mesma situação a forma não semântica é utilizando a tag <i>

Exemplo: Este pedaço está em i

Agora a forma semantica definida com a tag <EM>,

Exemplo: Este pedaço está em EM

Texto marcado

Podemos também criar textos marcados utilizando a tag <mark>

Texto deletado/taxado

O trecho a seguir será deletado coma técnica de taxado através da tag descontinuada <del>, texto deletado

Texto inserido ou sublinhado

podemos utilizar de forma visual a tag <u>, para sublinhar, assim como podemos de forma semântica e mais recomendada usar a tag <ins>, que possui um valor de sentido além do sublinhado

Exemplo:Este trecho utiliza a tag não semântica <u>

Exemplo:Este trecho utiliza a tag semântica <ins>

texto sobrescrito

Textos como a potenciâção na matemática utlilizam efeitos com o sobre escrito como o exemplo do 10 elevado a 20 se escreve com a tag <sup>

Exemplo:1020

Texto subescrito

Já em casos onde a letra ou texto fica na parte de baixo que é o efeito de subescrito se utiliza a tag <sub>, como na fórmula química da água o h2O

Exemplo:H2O
