Como Trabalar Com Vídeos

Vídeos de hospedagem própria

Neste caso tenho a tag <video> para indicar que o conteúdo é um vídeo. O src=""
indicando o caminho do arquivo, além disso, temos os controles de vídeo, com o
parâmetro controls, para colocar umagem de tumb usei o parmetro poster=""
os outros são loop, e até autoplay, que inicia o vídeo com o carregamento da
página. Por fim fiz uma pequena formatação na largura do vídeo com o parâme-
tro width="". Caso eu queira deixar outras opções de vídeos por questões de
compatibilidade devemos aninhar a tag <source>.
