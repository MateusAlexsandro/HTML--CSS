Trabalhando com links

Os links são relacionados a coisas como ancoras, links em computadores são clinks são hypertextos que permitem que com um clik eu seja redirecionado a outra página ou diretório web ou local.
A tag utilizada para pontar um link em um código HTML é a <a>

Links externos

caso o link seja de uma página externa, e quer que abra em outra guia, passe o parâmetro target com o valor ="_blank", e para semântica temos o parâmetro rel, que passamos o valor para indicar aos algoritomos que este link é do tipo externo ="External"
